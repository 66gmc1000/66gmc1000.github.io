Create an Azure GPU-backed Nvidia CUDA Worker Node via Azure CLI

June 14, 2021 One-minute read
blog
scripts • azure • oneliners • azcli • gpu • cuda

Login into Azure CLI as a service principal

az login \
--service-principal \
-u ${ARM_CLIENT_ID} \
-p ${ARM_CLIENT_SECRET} \
--tenant ${ARM_TENANT_ID}

Search for Azure virtual machines available in a specific region

az vm list-sizes -l eastus | grep Standard_N

Login with az service principal

az login --service-principal -u ${ARM_CLIENT_ID} -p ${ARM_CLIENT_SECRET} --tenant ${ARM_TENANT_ID}

List all accounts in subscription

az account list

Set the active Azure subscription

az account set --subscription ${ARM_SUBSCRIPTION_ID}

Create a new Azure resource group in the eastus region

az group create -l eastus -n play-ResourceGroup

Create an Azure Standard_NC6_Promo GPU virtual machine with SSH key authentication

az vm create -n s1-gVM1 -g play-ResourceGroup --image ubuntults --data-disk-sizes-gb 10 20 --size Standard_NC6_Promo --generate-ssh-keys

Get new public ip to ssh to in next command

vps1_ip=$(az vm show --resource-group play-ResourceGroup --name s1-gVM1 -d --query [publicIps] --output tsv | sed -e 's/,$//')

SSH to GPU node and install CUDA toolkit, then reboot

ssh force@$vps1_ip 'CUDA_REPO_PKG=cuda-repo-ubuntu1804_10.2.89-1_amd64.deb; wget -O /tmp/${CUDA_REPO_PKG} http://developer.download.nvidia.com/compute/cuda/repos/ubuntu1804/x86_64/${CUDA_REPO_PKG}; sudo dpkg -i /tmp/${CUDA_REPO_PKG}; sudo apt-key adv --fetch-keys https://developer.download.nvidia.com/compute/cuda/repos/ubuntu1804/x86_64/7fa2af80.pub; rm -f /tmp/${CUDA_REPO_PKG}; sudo apt-get update; sudo apt-get install cuda-drivers -y; sudo reboot'

Destroy resource group and vms

az group delete --name play-ResourceGroup
